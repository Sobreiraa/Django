Criando um projeto com SQLite3 e sua estrutura:
	
	DJANGO1
	
	*manage.py - Arquvio que vamos usar para executar comando do framework django.
	*__init__.py - Transforma um diretório em um pacote no python.
	*asgi.py - Usado para fazer ligação entre o django e um servidor web.
	*settings.py - Configurações django para o seu projeto.
	*urls.py - Arquivo de rotas.
	*wsgi.py - Arquivo exclusivo para nossas publicações.

	MIGRATIONS
	
	*admin.py - Arquivo para adicionar modelos para que reflita na adminstração do django
	*apps.py - Arquivo para os apps
	*models.py - Criar os modelos de dados
	*tests.py - Arquivo para criar testes.
	*views.py - Arquivo para funções de rotas para abrir os templates de visualização
